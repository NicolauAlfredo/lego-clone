# LEGO CLONE

This project is part of the **Web Developer Course by [Develhope](https://www.develhope.co/)** team exercise. The goal is to clone the LEGO website as closely as possible, practicing front-end development skills using **HTML**, **CSS** and **JAVASCRIPT**.  

## Project Overview
- Team project exercise for learning web development from scratch.
- Currently implemented using **HTML** and **CSS**.
- The project will later expand to include **JavaScript**, **React**, and **TypeScript** for dynamic features and modern web interfaces.  

## Current Technologies
- HTML
- CSS

## Future Improvements
- Implement responsive design
- Add JavaScript interactions
- Explore React and TypeScript in later stages of the course
